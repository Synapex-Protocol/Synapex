How It Works

From Idea to Deployment

synapex-lab — workflow 
$ synapex lab init --project "face-recognition-v2"

✓ Project created: face-recognition-v2
✓ GPU allocation: 4x RTX 4090 via compute marketplace
✓ Base model loaded: synapex/vision-backbone-v1

$ synapex train --config train.yaml --epochs 120

► Training started on distributed cluster
  Epoch  032/120  |  Loss: 0.0234  |  Acc: 98.7%  |  ETA: 2h 14m
  ...
✓ Training complete. Model saved: models/face-rec-v2.pt

$ synapex publish --module face-rec-v2 --license MIT

✓ Module published to SYNAPEX Module Library
✓ IP registered on-chain: 0x7a3f...c4e2
✓ Now earning $SYNX per inference call
